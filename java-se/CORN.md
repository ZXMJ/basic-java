#秒(允许的通配符[, - * /])
5,16:指定5秒,16秒

5-16:从5秒到16秒

*:每秒

15/5:从5秒开始,每隔5秒执行

#分(允许的通配符[, - * /])
同上
#小时(允许的通配符[, - * /])
同上
#日(允许的通配符[, - * / L W])
, - * /用法同上

L每月最后一天

5W工作日:每月5号最近的工作日
#月(允许的通配符[, - * /])
同上上
#周(允许的通配符[, - * / L #])
, - * /用法同上

5L:本月的最后一个星期五

10#5:第10周的星期五
#年(允许的通配符[, - * /])非必填
